EMBEDDIA Toolkit API wrapping EMBEDDIA API-based Services

# Running the Docker

```
docker-compose pull

docker-compose up
```

# Dev Setup

**Clone with EMBEDDIA Submodules**

```
git clone --recursive https://git.texta.ee/texta/embeddia-toolkit.git
```

**Building & Pushing Images**

```
docker-compose build

docker-compose push
```
